プログラミングとは

プログラミングとは、人がコンピューターを思い通りに動かすために必要となる、コンピューター専門の用語のことです。

1950年代に初めて生み出されて以来、現在では1,000以上ものプログラミング言語の種類が存在していると言われています。

プログラミング言語がなぜ生まれてきたかいうと、以下のように人とコンピューターでは理解できる言語に違いがあるからです。

コンピューターを動かしている心臓部であるCPU が理解できるのは、数字の「0」と「1」の羅列だけです。

人がコンピューターの実行させたい命令があっても、「0」と「1」だけでは果てしない数の組み合わせが必要となり、細かい指示を出すのは極めて困難となってしまいます。

プログラミングを学習する上で重要な考え方

◯問題解決能力

◯基礎を固める

◯実践と通じて学ぶ

◯疑問点まず調べる

◯コードの可読性を重視する

◯継続的な学習を心がける

HTML

HTML(HyperText Markup Language)はウェブページの骨格を形成するマークアップ言語で、ウェブコンテンツの構造を定義します。HTMLタグを使用してテキスト、画像を配置し、ウェブページの基本的なレイアウトを作成します。

CSS

CSS(Cascading style sheet)はHTMLで作成されたウェブページのスタイルを指定するための言語です。フォントの種類、色、間隔、レイアウトの調整など、ページのビジュアルデザインを制御します。CSShaページの外観を改善し、ユーザー体験を向上させます。

GIT

GITは、ソースコードのバージョン管理システムであり、複数の開発者がプロジェクトに協力する際のコードの変更を追跡し、管理するために使用されます。効率的なブランチ管理、マージ機能を提供し、開発プロセスをスムーズにします。

JavaScript

JavaScriptはWebページに動的な機能を追加するプログラミング言語です。ユーザーインタラクションの処理、動的なコンテンツの表示、フォーム検証など、ウェブアプリケーションのインタラクティブな要素を実装します。ブラウザ上で直接実行されるため、即座に反応するユーザーインターフェースを提供します。

MySQL

MySQLは、リレーショナルデータベース管理システム(RDBMS)であり、ウェブアプリケーションのデータを格納、管理するために使用されます。SQL(Structured Query Language)を使用してデータの検索、挿入、更新、削除を行い、PHPなどの言語と組み合わせて動的なコンテンツの生成に貢献します。

PHP

PHP(Hypertext Preprocessor)は、サーバーサイドで実行されるスクリプト言語で、動的なウェブページやアプリケーションの開発に使用されます。データベースとの連携、セッション管理、コンテンツの動的生成など、サーバー側で処理を担います。
